# gamboo UI Agent Benchmark — 評価基準

> AI が生成した UI を評価するための rubric。各項目 1-5 点。

---

## 1. gamboo Fidelity（DS 準拠性）— 配点 30%

| スコア | 基準 |
|--------|------|
| 5 | 全てのコンポーネントが DESIGN.md の Quick Reference 通り。semantic color 使用。禁止パターンゼロ |
| 4 | 90% 準拠。1-2 箇所の軽微な逸脱（spacing の微差等） |
| 3 | 70% 準拠。semantic color は使用しているが、一部ハードコードカラーあり |
| 2 | 50% 準拠。構造は gamboo 的だが、禁止パターン複数あり |
| 1 | gamboo UI のルールをほぼ無視 |

### チェック項目
- [ ] `primary-*` を使用（`bg-blue-*` / `bg-indigo-*` 不使用）
- [ ] `text-body` / `text-slate-900` を使用（`text-black` 不使用）
- [ ] `shadow-sm` / `shadow-md` のみ（`shadow-lg` 以上は overlay 限定）
- [ ] `rounded-xl` on cards（`rounded-none` 不使用）
- [ ] カラーバー（`border-t-4` / `border-l-4`）不使用
- [ ] ボタンサイズ `h-8` / `h-10` / `h-12`（`py-0.5` 不使用）
- [ ] `cursor-pointer` on interactive elements

---

## 2. Craft（デザイン品質）— 配点 25%

| スコア | 基準 |
|--------|------|
| 5 | Linear / Notion 級の洗練度。余白・タイポグラフィ・アイコン全てが意図的 |
| 4 | プロダクション品質。ほぼそのまま出荷できる |
| 3 | 機能的だが、余白やアラインメントに粗さがある |
| 2 | 動くが「AIが作った感」が残る（均一な余白、テンプレート感） |
| 1 | 粗い。構造が崩れている |

### チェック項目
- [ ] 4px グリッド準拠のスペーシング
- [ ] テキスト階層（見出し / 本文 / サブテキスト）が明確
- [ ] 余白が「呼吸している」（窮屈でも空きすぎでもない）
- [ ] アイコンが適切なサイズ・色で使用されている
- [ ] 1 画面 3 色以内

---

## 3. Usability（ユーザビリティ）— 配点 20%

| スコア | 基準 |
|--------|------|
| 5 | ユーザーが迷わない。情報階層・操作フロー・フィードバック全て適切 |
| 4 | ほぼ直感的。軽微な改善点あり |
| 3 | 使えるが、一部の操作がわかりにくい |
| 2 | 混乱する箇所が複数ある |
| 1 | 操作困難 |

### チェック項目
- [ ] 主要アクション（CTA）が明確
- [ ] 空状態・エラー状態の考慮
- [ ] フォームのラベル・バリデーション
- [ ] 読み込み状態の考慮

---

## 4. Accessibility（アクセシビリティ）— 配点 15%

| スコア | 基準 |
|--------|------|
| 5 | WCAG 2.1 AA 完全準拠。aria, keyboard, contrast 全て対応 |
| 4 | 主要な a11y は対応。軽微な漏れあり |
| 3 | コントラスト・ラベルは OK だが、keyboard / aria が不十分 |
| 2 | 基本的なラベルのみ |
| 1 | a11y 未考慮 |

### チェック項目
- [ ] `<th scope="col">` on tables
- [ ] `aria-label` on icon-only buttons
- [ ] `aria-current="page"` on active nav
- [ ] `role="dialog"` + `aria-modal` on modals
- [ ] フォーカスリング（`focus:ring-2 focus:ring-primary-500/50`）
- [ ] 色だけでない情報伝達

---

## 5. Rule Compliance（ルール遵守）— 配点 10%

| スコア | 基準 |
|--------|------|
| 5 | `npm run design:check` で検出される違反がゼロ |
| 4 | 1-2 件の warn レベル違反 |
| 3 | 3-5 件の違反（error 含む） |
| 2 | 6-10 件の違反 |
| 1 | 10 件超の違反 |
