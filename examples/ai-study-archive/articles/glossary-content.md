# 用語集（用語集ページ用コンテンツ）

| 用語 | 説明 |
|---|---|
| MCP（Model Context Protocol） | AIが外部のツールやデータに安全にアクセスするための共通の接続規格 |
| デザイントークン | 色・サイズ・余白などの値に名前を付けて一元管理する仕組み |
| PRD | Product Requirements Document。何を作るかをまとめた要件定義書 |
| コンポーネント | UIを構成する再利用可能な部品の単位（ボタン、カードなど） |
| WSL2 | Windows上でLinux環境を動かす仕組み |
| Astro | Markdownファイルなどから静的サイトを自動生成するツール |
| Vite | フロントエンド開発を高速に行うためのビルドツール |
| Tailwind CSS | クラス名を組み合わせてスタイルを当てるCSSの仕組み |
| Content Collections | Astroが持つ、Markdownなどのコンテンツをまとめて管理する機能 |
| フロントマター | Markdownファイルの先頭に書く、タイトルや日付などのメタ情報 |
