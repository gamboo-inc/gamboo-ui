Figmaなどデザインツールの新機能（2026年7月）

更新日: 2026/07/10 ・ 難易度: 初級

概要

03_情報収集 で日次収集していたFigma関連ニュースの時系列まとめ。6月開催のFigma Config 2026で発表された機能（Motion、Code Layers、3D/シェーダーなど）が7月にかけて順次ロールアウトしている状況を追った記録。

2026-07-06

  • Figma Config 2026で大型発表 — キャンバス上でモーション・3D・コード・テクスチャを扱える新機能群を一挙公開。Figma Blog: Config 2026 Recap
  • Figma Motion — タイムラインまたはFigmaエージェントで精密なアニメーションを作成し、再利用可能なモーションシステムを構築、開発者向けにそのまま書き出し可能に。Figma Config 2026 Recap
  • 3D Transforms / Custom Shaders — デザインや画像に立体的な奥行きを付与でき、さらに自然言語指示でWebGPUシェーダーをエージェントが生成する機能も追加。uWarp: Figma Config 2026 Recap
  • Code Layers（コードレイヤー）ロールアウト開始 — デザインやコードベースを起点にインタラクション案をプロンプトで生成・比較できる機能が2026年7月から展開開始。Figma Config 2026 Recap
  • Weaveツール群によるAI画像生成 / デスクトップのタブグループ機能 — 背景差し替えやロゴ追加などの画像編集をファイル内で完結でき、デスクトップアプリではタブをグループ化・色分け可能に。Figma Help Center: What’s new from Config 2026

2026-07-07

  • Figma Motion — タイムライン、またはFigmaエージェントへのプロンプト操作で精密なアニメーションを作成し、開発可能な状態でそのまま書き出せる新機能。Config 2026: New Materials, New Tools and a More Expressive Canvas
  • Code Layers（コードレイヤー） — デザインレイヤーをワンクリックまたはプロンプトでインタラクティブな「コードレイヤー」に変換できる新機能。GitHubリポジトリをキャンバス上に読み込み、React/npmパッケージを使った実装をチーム全員でリアルタイム編集・同期可能。2026年7月からアーリーアクセス開始（config-betasでウェイトリスト登録）。Figma Launches Code Layers & Motion at Config 2026
  • 3D変形・シェーダー機能 — デザインや画像に立体感・質感を与え、モーションと組み合わせてリアルな質感の動きを表現可能に。シェーダーはプロンプトでエージェントが生成。Figma Config 2026: Code, Motion, and AI Tools
  • Figmaエージェントの「Skills」機能拡張 — チームのワークフローや規約を再利用可能な指示セット（Skills）としてパッケージ化し、コミュニティやチーム間で共有できるように。What’s new from Config 2026 – Figma Learn
  • Weaveツール統合 — 背景差し替え、ロゴ追加、アスペクト比変更などのAI画像生成タスクをFigma Designから離れずに実行できるツール群が追加。Figma Config 2026: Code, Motion, and AI Tools

2026-07-08

  • Config 2026で複数の新機能を発表 — モーション、奥行き、テクスチャ、コード機能がキャンバス上で統合。Figma Blog
  • Figma Motion — タイムラインやFigmaエージェントで精密なアニメーションを作成し、開発可能な状態まで仕上げられる新機能。uinkits
  • Code Layers（7月ロールアウト開始） — デザインレイヤーをワンクリック/プロンプトでインタラクティブなコードレイヤーに変換可能。Figma Help Center
  • 3D変形・シェーダーエフェクト — デザインや画像に立体感を付与し、モーションと組み合わせてリアルな動きを表現。エージェントによるシェーダー自動生成も可能に。Figma Blog
  • Weaveツール統合・エージェントのFigJam/Slides拡張 — AI画像生成（背景差し替え、ロゴ追加等）や、FigJam・Slidesでの共同編集エージェントが利用可能に。Figma Blog

2026-07-09

  • AI画像編集の強化（2026年7月7日） — ツールバーから複数のAI画像編集を同時生成でき、生成中も並行して作業を継続可能に。Figma product news and release notes
  • Code Layers（2026年7月ロールアウト開始） — デザインまたはコードベースからインタラクションのアイデアをプロンプトで作成・比較でき、Xcodeとも連携してモバイルデザインを直接プレビュー可能。Config 2026: New Materials, New Tools and a More Expressive Canvas
  • Figma Motion / 3D変形 / シェーダー — タイムラインまたはFigmaエージェントで精密なアニメーションを作成。3D変形やAIによるシェーダー生成も追加され、質感表現が可能に。Figma Config 2026: Code, Motion, and AI Tools
  • Figmaエージェントの拡張（Skills） — ワークフローやコーディング規約を再利用可能な「Skills」としてパッケージ化し、Notion・Slack・GitHub・Atlassianなど既存ツールとも連携可能に。Everything announced at Config 2026 - Figma Forum

2026-07-10

  • Config 2026（6/24開催）でCanvas刷新を発表 — モーション、3D変形、シェーダー、コードレイヤーがキャンバス上に統合。Figma Blog
  • Figma Motion — タイムライン・キーフレーム・プリセット付きでモーション制作がキャンバス内で可能に。
  • Code Layers（コードレイヤー） — デザインレイヤーをワンクリックでインタラクティブなコードレイヤーに変換、7月からロールアウト開始（ウェイトリスト経由）。uwarp.design
  • Weave Tools — 背景差し替え・ロゴ追加・アスペクト比変更などのAI画像生成機能をファイル内で完結。johnrodrigues.substack.com
  • 並列AI画像編集（7/7リリース） — ツールバーから複数のAI画像編集を同時生成し、読み込み中も作業継続可能。blog.mean.ceo

次にやること

Code Layersのウェイトリストに登録し、アーリーアクセスが来たらmelta-uiのコンポーネントで試す。
