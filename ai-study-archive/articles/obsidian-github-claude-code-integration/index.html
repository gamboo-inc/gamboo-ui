AI活用ナレッジ — ObsidianとGitHub/Claude Code連携

更新日: 2026/07/06 ・ 難易度: 初級

概要

ObsidianのVault（MyNotes）をGitで管理し、GitHubにバックアップする環境を構築した記録。

環境

  • WSL2 + Ubuntu を構築
  • Claude Code、Codex を導入
  • GitHub、Figma と連携
  • VS Code を導入

実施したこと

1. Git / GitHub連携の基本設定

  • Obsidianの保管庫（MyNotesフォルダ: /mnt/c/Users/hiros/Documents/obsidian/MyNotes）をGitリポジトリとして初期化
  • .gitignore を作成し、以下を除外設定
    • .obsidian/workspace.json など、PC固有で頻繁に変化するファイル
    • .DS_Store、Thumbs.db などOS生成ファイル
    • （プラグイン設定 .obsidian/plugins/*/data.json は、複数環境で共有したいため除外せず含める方針に変更）
  • Git identityを設定（名前: hiro444647、メール: GitHub提供のnoreplyアドレス）
  • GitHubに非公開（private）リポジトリ hiro444647/MyNotes を作成し、push完了

2. obsidian-gitプラグインで自動バックアップ化

  • Vault内に既にインストールされていた obsidian-git プラグインを有効化
  • 設定内容（data.json）:
    • autoSaveInterval: 15 → 15分ごとに変更があれば自動でadd+commit+push
    • disablePush: false → バックアップ時に自動でpushも実行
    • autoPullInterval: 0 → このPCのみで使うため自動pullは無効
  • Windows側にもGit for Windowsをwingetでインストール（Obsidian本体がWindows上で動くため必要）
  • Windows側のGitにも同じIdentityを設定
  • GitHub認証はVS Code拡張機能の「Allow」操作で完了（Git Credential Manager経由）

3. Vaultのフォルダ構成

初心者向けにシンプルな3分類で開始:

  • 01_デザイン: デザインの仕事のアイデア・参考サイト・配色・スケッチなど
  • 02_作業記録: Claude Codeとのやり取りや作業ログ（日付ごとにファイル作成）
  • 03_情報収集: 調べた情報・学んだことのストック

4. 今後の運用

  • 通常はobsidian-gitプラグインが自動でGitHubに同期（15分ごと）
  • 手動で今すぐ同期したい場合はObsidian左サイドバーのGitアイコンから操作可能
  • 新しいノートを作るときは、右クリック→「新規ノート」で作成し、自動保存される

GitHubの料金・容量について（調査メモ）

  • GitHub Freeプランで、非公開リポジトリは無制限に作成可能。料金は発生しない
  • リポジトリサイズの推奨上限は1GB未満、強く推奨されるのは5GB未満（ハード上限100GB）
  • ファイル単体では100MBを超えるとpushがブロックされる（Git LFSが必要）
  • テキスト中心のObsidianノートであれば、容量・料金ともにほぼ心配不要

Claude Codeプラグインについて（調査メモ）

  • Anthropic公式マーケットプレイス claude-plugins-official から様々なプラグインを導入可能
  • /plugin コマンドで一覧表示・インストールができる
  • 代表例: claude-code-setup（設定提案）、security-guidance（セキュリティレビュー自動化）、code-review（PRレビュー）、commit-commands（コミット・push簡略化）
  • これらのプラグインはClaude Code自体の拡張機能であり、Obsidian側のGit管理設定とは独立していて干渉しない

つまづいたポイントと解決策

  • 新しいVS Codeウィンドウで claude コマンドが見つからないエラーが発生 → 原因: PowerShell（Windows側）のターミナルで実行していたため → 解決: 「Recent」からWSL接続（hiros [WSL: Ubuntu]）経由でMyNotesフォルダを開き直し、WSL側のターミナルでClaude Codeを起動
